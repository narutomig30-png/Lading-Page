Por favor, siga estes passos para acessar seu site corretamente:

1. Verifique se o XAMPP está em execução
Abra o painel de controle do XAMPP e verifique se os serviços Apache e MySQL estão iniciados (eles devem estar com o fundo verde).

Se não estiverem iniciados, clique nos botões "Start" para cada um.

2. Use a URL correta no navegador
Com o XAMPP em execução, abra seu navegador de internet (Chrome, Firefox, etc.) e digite o seguinte endereço na barra de URL:

http://localhost/LANDING-PAGE/

Pressione Enter.
